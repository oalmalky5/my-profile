Projects

NexDIY Fullstack Application

NexDIY is a robust full-stack blogging platform crafted specifically for DIY enthusiasts. Built using the MERN stack, this application boasts an intuitive user interface paired with a comprehensive backend. Users can seamlessly register, login, create, update, and delete posts, while also having the ability to comment and interact with the DIY community. Its holistic design ensures that DIY aficionados have a streamlined experience to share and discuss their creative projects.

Mauzoun Responsive Website

The redesigned website for Mauzoun, a renowned content writing and publishing house, showcases a modern and efficient design. Developed using a tech stack that includes Next.js, React.js, JavaScript, CSS, and SASS, the platform offers users an intuitive and responsive browsing experience. The site ensures smooth navigation across various devices and integrates dynamic animations and transitions, balancing aesthetics with the latest technology. This website is a testament to the blend of Mauzoun's content expertise and high-caliber web development, emphasizing both functionality and user engagement.

Jawlat Jeddah Tourism App

The "Jawlat Jeddah" project is a frontend template designed to showcase Jeddah's tourist spots. Built using React, JavaScript, and CSS, it lays the groundwork for a full-fledged tourism app, emphasizing Jeddah's unique attractions. With its modern web development techniques, streamlined design, and organized layout, it offers developers a robust base for crafting an engaging digital platform, bridging potential visitors with Jeddah's rich cultural offerings.
